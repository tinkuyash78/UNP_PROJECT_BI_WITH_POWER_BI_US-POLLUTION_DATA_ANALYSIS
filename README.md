# UNP_PROJECT_BI_WITH_POWER_BI_US-POLLUTION_DATA_ANALYSIS

I completed a US Pollution Data Analysis project in collaboration with UNP using Power BI Desktop to evaluate historical pollution trends and atmospheric health indicators. The dashboard provides a data-driven analysis of US air quality from 2000–2016, focusing on

 pollutants. I designed an interactive dashboard featuring high-level KPIs, including a "Worst AQI" peak of 218 and long-term average trends. Using advanced DAX measures, the report tracks how pollutants like Carbon Monoxide and Sulphur Dioxide have significantly decreased over sixteen years. Users can filter by State, City, and Date to uncover regional hotspots, such as high Ozone levels in San Bernardino. The project includes a category-wise breakdown (Good to Hazardous) and Top/Bottom 10 rankings for various counties. I utilized quarter-wise analysis to identify seasonal volatility in air quality across the country. The dark-themed UI enhances readability, while seamless navigation allows for deep-dives into specific gas metrics. This repository demonstrates my proficiency in data modeling, geographic visualization, and temporal trend analysis. It serves as a professional tool for transforming complex EPA monitoring data into actionable environmental insights.
